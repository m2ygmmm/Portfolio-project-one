My projects

Presenting a selection of my latest projects, each accompanied by its source code available on GitHub. I welcome and highly appreciate any feedback you may have. Feel free to explore and share your thoughts!

Mixed messages

A simple JavaScript program

A simple JavaScript program that can predict what will happen to you during your week.....Written in Vanilla JavaScript using a variation of methods and objects to output a made up and random string!

NASA API

A Website to search through NASA's Image Library

A simple responsive website to search for images on the NASA Image library using the free to use NASA API written in HTML, CSS (with the Bootstrap framework) and vanilla JavaScript. Enter a 'space' related term into the search bar and hit search to see the results! Filter through the results using the filter button.

Jammming

A React program for Spotify

A simple React application that allows users to search for a track and add it to a playlist and send it straight to their spotify account. The application was designed using the Spotify API and various React functions and components

FoodHelper

Fighting food poverty!

Developed using NodeJs, ExpressJs, Handlebars and MongoDB, FoodHelper is a Web Application which aims to fight food poverty by providing a platform for charities and organizations To share what they can provide to you.
